How To Run??

After Starting Apache and MySQL in XAMPP, follow the following steps

1st Step: Extract file named ELM
2nd Step: Copy the main project folder
3rd Step: Paste in xampp/htdocs/

Now Connecting Database

4th Step: Open a browser and go to URL “http://localhost/phpmyadmin/”
5th Step: Then, click on the databases tab
6th Step: Create a database naming “elm” and then click on the import tab
7th Step: Click on browse file and select “elm.sql” file
8th Step: Click on go.

After Creating Database,

9th Step: Open a browser and go to URL “http://localhost/elm/elm/” [by default open user side]

Admin Panel path: http://localhost:/elm/ELM/admin/login.php
User Panel path: http://localhost:/elm/ELM/login.php
